Typography

The following eight pieces each show the information of "Bharti Kher: The Body Is a Place" using all eight typography systems we learned in week 4.

This is the axial system, it puts text along axes. While the axes can be rotated, the text always follows the axis.

This is the radial system, texts come out of a shape or line like sun rays.

This is the dilatational system, text looks rounded like wrapped along a circle This can lead readers to look at the text in a certain direction.

This is the random system, like the name suggests it's all random. No rules, no guidelines, just go wild with it.

This is the grid system, texts follow a grid. You can design the grid in any way, but the text must be contained within the grid.

This is the modular system, similar to the grid system, but it got fewer restrictions. Text can be placed anywhere as long as they are in different blocks, like a module.

This is the transitional system, it might look similar to the dilational system, but instead, the curves can be twisted, looking more like a hilly mountain than a circle.

This is the bilateral system, text can be placed in any way you like as long as it is centre aligned.
